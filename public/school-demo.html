📖 Kaise Use Karein
Principal और Parent दोनों के लिए step-by-step guide
🏫 Principal के लिए
1 School Register करें
"Principal" tab select करें, अपना नाम और phone डालें, फिर "शुरू करें" पर click करें। School का नाम, city, fee भरें।
पहले से register हैं? Same phone से login करें - school auto-load हो जाएगा!
2 Student जोड़ें
Dashboard पर "नया Student जोड़ें" button दबाएं। Child का नाम, parent phone, class (Nursery/LKG/UKG) भरें।
Roll number automatic बनेगा! Parent का phone डालने से वो login कर सकेंगे।
3 Daily Diary लिखें
हर रोज "Daily Diary" button से class select करें, सभी students का mood, meals, nap एक साथ भरें।
Parents को तुरंत दिखेगा! आप phone से 30 sec में सब भर सकते हैं।
4 Photos & Alerts भेजें
Class activities की photos upload करें, ज़रूरत पर emergency alert भेजें - सब parents को instant।
"Share Link" button से WhatsApp पर school का link भेजें - parents भी app install कर सकेंगे!
👨‍👩‍👧 Parent के लिए
1 Phone से Login करें
"Parent" tab select करें, अपना phone number डालें, "मेरे बच्चे खोजें" दबाएं।
OTP नहीं चाहिए! बस phone number डालो - school ने जो phone दिया है, same डालो।
2 बच्चे का Diary देखें
बच्चे पर click करें - आज की diary, पिछले हफ्ते का summary, class photos सब दिखेगा।
हर रोज़ school से update मिलेगा - मूड, खाना, nap सब!
3 Pickup OTP लें
जब बच्चे को pick-up करना हो, "Pickup OTP" button दबाएं - daily 4-digit OTP मिलेगा।
Driver/Teacher को OTP बताएं - safe pickup guaranteed!
4 Chat & Feedback
"Chat" tab से school से directly बात करें, "Feedback" tab से rating/suggestion दें।
कोई भी problem हो - Principal को instant message भेजें!
💡 Install Tip
पहली बार "Install" button दिखेगा - दबाकर phone में install कर लो। बाद में offline भी चलेगा!
