⚡ Quick Summary (5 min setup)

  1. YouTube Studio pe Go Live karo → Stream Key lo
  2. OBS Studio me YouTube RTMP URL + Stream Key paste karo
  3. OBS me Start Streaming click karo
  4. YouTube Live URL DainikState me add karo
  5. TV Channel khole → Auto-play hoga! 🎉
1
📥 OBS Studio Install Karo

Download from: obsproject.com

Available for: Windows / Mac / Linux — 100% FREE

# After install, open OBS Studio
# First time: Auto Configuration Wizard use karo
2
📺 YouTube Studio me Live Stream Setup

Browser me jao:

studio.youtube.com → Go Live

Steps:

  • "Go Live" button click karo (top right)
  • "Stream" tab select karo (not Webcam)
  • Title daal do: e.g., "DainikState Live News"
  • Privacy: Public (ya Unlisted)
  • Click "Create Stream"

YouTube dega 2 cheezein:

Stream URL: rtmps://a.rtmp.youtube.com/live2
Stream Key: xxxx-xxxx-xxxx-xxxx-xxxx

⚠️ Stream Key SECRET hai — kisi ko share mat karo!

  • Service: YouTube - RTMPS
  • Stream Key: YouTube wala paste karo
  • OK click karo

▶️ Start Streaming

OBS me:

  1. "Start Streaming" button click karo (right side)
  2. YouTube Studio pe "Go Live" → "Stream" → confirm
  3. YouTube me 5-15 sec me live ho jayega

YouTube URL note karo:

https://www.youtube.com/watch?v=XXXXXXXXX

Ye URL DainikState me add karna hai ⬇️

6
📺 DainikState me Add Karo

Option A: news.html form se

  • Open /news.html
  • Header me ⚙️ → "Add News to Channel"
  • Title: e.g., "🔴 LIVE: DainikState News Broadcast"
  • Video URL: YouTube Live URL paste karo
  • Featured: ✅ ON karo (top pe dikhega)
  • Submit

Option B: Direct API call (advanced)

curl -X POST "https://canyou-uqkp.onrender.com/api/news/group-channel/add" \
  -H "Content-Type: application/json" \
  -d '{
    "title": "🔴 LIVE: DainikState News",
    "videoUrl": "https://www.youtube.com/watch?v=XXX",
    "category": "news",
    "groupId": "dainikstate",
    "featured": true
  }'
7
🎉 TV Channel me LIVE Dekho!

Open: /news-channel.html

  • ✅ Featured news = LIVE video top pe dikhega
  • ✅ "🔴 LIVE NOW" badge automatically show hoga
  • ✅ Real-time viewer count
  • ✅ Auto-play (muted by default)
  • ✅ Click 🔊 for sound
  • ✅ Fullscreen available

Pro Tips (Broadcast Quality)
  • Resolution: 1920x1080 (Full HD) for best quality
  • Bitrate: 4500-6000 kbps for 1080p
  • FPS: 30 fps (smooth), 60 fps (sports)
  • Encoder: x264 (CPU) ya NVENC (NVIDIA GPU)
  • Audio: 128 kbps AAC stereo
  • Internet: Min 10 Mbps upload speed

OBS Best Settings:

Alternative: Facebook Live
